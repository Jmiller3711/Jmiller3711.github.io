NCAA March Madness Bracket Builder

This tool allows you to load or input data for a specific year's NCAA Tournament. Select the year and statistic you would like to process, then let's get ready to "RUMBLE".
For example, if you want to see which team from 2016 would win based on their BPI (Basketball Power Index), select 2016, BPI, then press the "RUMBLE" button.
If you would like to enter your own data to the bracket, first select a year to load team names, enter data for all 64 teams, then press "RUMBLE".
The "Reset" button will clear all input fields.
